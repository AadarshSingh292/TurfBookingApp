# TurfBookingApp

A console-based Java application for booking sports turf slots with authentication, booking management, and payment handling features.

## Project Structure

```text
TurfBookingApp/
└── src/
    ├── model/
    │   ├── User.java
    │   ├── Sport.java
    │   ├── Football.java
    │   ├── Cricket.java
    │   ├── Slot.java
    │   ├── Booking.java
    │   └── Payment.java
    │
    ├── service/
    │   ├── AuthService.java
    │   ├── BookingService.java
    │   └── PaymentService.java
    │
    ├── util/
    │   ├── InputValidator.java
    │   └── ConsoleUI.java
    │
    └── TurfBookingApp.java
```

## Features

- User Authentication
- Turf Slot Booking
- Payment Management
- Input Validation
- Console-Based UI
- Object-Oriented Design

## Technologies Used

- Core Java
- OOP Concepts
- Collections
- Exception Handling

## How to Run

Compile and run:

```bash
javac TurfBookingApp.java
java TurfBookingApp
```

Main entry point:
- `TurfBookingApp.java`
